# Omarchy-menu-MultiKeybindings

An expansion to omarchy-menu-keybindings
- Adds an unified menu for keybinds on various applications

## Currently supported
- Ghostty

## Usage
Move every scripts to ~/.local/bin

You might wanna replace your keybind menu in .config/hypr/bindings.conf

```
unbind = SUPER, K
bindd = SUPER, K, Keybinding Menu,exec, omarchy-menu-multikeybindings
```
